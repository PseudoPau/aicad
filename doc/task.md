# 任务
- 搭建AI CAD的环境（freecad, cadquery, openscad等）和工作流环境（如果需要）
- 用多模态输入，用AI基于参考图片制作一个立体仓库模型，要求
  - 尺寸、数量等可以调节和联动
  - 部件要作为独立的模型文件被创建和保存。货架要引用这些部件组装出来，而不要在货架中直接用box等命令做纯造型体
  - 各部件的组装关系、和上述参数调整的联动都正确。尤其是一些部件搭接细节，要和图片输入一致
  - 将上述生成能力封装成一个小工具，可被人简单操作，或AI调用
  
# 交付物
- 最终生成的模型在不同参数值下的若干截图、部件搭接细节和图片一致的展示
- 人工操作界面或AI调用的接口入参说明
- 你在图片选择、prompt、工作流等方面，采取了哪些措施提升最终效果
- 你采取了哪些措施让AI更理解部件之间的装配关系，尤其是搭接细节？
